# OpenClaw Source Architecture Survey

This repository contains a repo-wide architecture survey of the OpenClaw source code.

## Files

- `openclaw-source-architecture-survey.md` — full system-level survey of the OpenClaw codebase

## Scope

The survey covers:

- the main OpenClaw technical framework
- the role of each major subsystem
- the plugin SDK and extension boundary
- the gateway, agent runtime, sessions, routing, and hooks
- bundled extensions, UI, native apps, and their relationships to the control plane
